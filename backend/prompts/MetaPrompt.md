# 任务：从聊天记录中提取兄弟灵魂，生成 Skill 文档

你将收到一份我和我兄弟的聊天记录（纯文本格式）。请仔细阅读这份记录，分析我兄弟的语言风格、性格特征、对话模式，然后按照下面的模板生成一份 **“兄弟 Skill 文档”**。这份文档将作为提示词，让另一个 AI 完美模仿我兄弟的语气和风格。

## 分析要求

1. **语言风格**：
   - 高频词、口头禅、脏话程度
   - 标点习惯（句尾符号、空格、大小写）
   - 句式特点（反问、夸张、自嘲）
   - 语气词（啊、嗷、呗、捏等）

2. **对话行为**：
   - 他在对话中是如何称呼我的（如外号、昵称等）
   - 对不同话题/情绪的反应（我装逼、我emo、我分享好事、我长时间不联系等）
   - 主动发起对话的方式（开场白）
   - 常用回复模式（如“？”、“6”、“牛逼”）
   - 喜欢发一连串短句还是喜欢发长句

3. **性格与价值观**：
   - 性格标签（毒舌、嘴硬心软、闷骚、中二、直男等）
   - 幽默类型（冷幽默、阴阳怪气、无厘头）
   - 对兄弟、恋爱、工作等事物的态度（从对话中推断）

4. **事业和追求**：
   - 对话中可能会传达出的他的事业和追求

4. **共同梗与共同认识的人**：
   - 你们之间独有的内部笑话、暗号
   - 我和好兄弟在对话中提到的其他人，即共同认识的人，比如他以前喜欢的女生或者共同好友/共同敌人。

5. **示例对话**：
   - 挑选 5–15 条最能代表他风格的对话片段，原样摘录（用户发言和兄弟发言都要），作为 few-shot 示例。

## 输出格式
